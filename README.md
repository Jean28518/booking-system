# booking-system

## How to run in production

```bash
git clone https://github.com/Jean28518/booking-system.git
cd booking-system
docker-compose up -d
```

## How to develop

```bash
sudo apt-get install python3-venv python3-dev
cd src/django
python3 -m venv .env
```

Now you can run the `run_development.sh` file.
